Hallo :-)

Das hier ist eine kleine Experimentierseite für ein - vielleicht - ganz hilfreiches OER-Tool.

Worum geht es?

Wenn man sich von Web-Diensten wie z.B. Deepl (ein Übersetzungsprogramm), eine eingetragene Übersetzung kopiert, dann erscheint im Text den man einfügt 'übersetzt mit Deepl'. Vor dem Einfügen ist dieser Textbaustein nicht sichtbar.

Was hat das mit OER zu tun?

Lambert Heller hatte die - wie ich finde - sehr coole Idee, ob man dieses Prinzip nicht vielleicht für OER nutzen kann: Wenn ich einen CC-lizenzierten Text kopiere und woanders einfüge, wird automatisch am Ende die Lizenz ergänzt.

Wie würde das funktionieren?

Ich habe ein bißchen recherchiert und so etwas scheint einfacher zu sein, als gedacht.

Umgesetzt wird das Ganze mit einem kleinen Javascript-Code-Schnipsel. (Der Text, der angezeigt werden soll, kann natürlich beliebig geändert/ angepasst werden.)

document.addEventListener('copy', (event) => { const pagelink = `\n\nVeröffentlicht von Nele unter CC BY 4.0 - Quelle: ${document.location.href}`; event.clipboardData.setData('text', document.getSelection() + pagelink); event.preventDefault(); });

Probiere es aus!

Probiere es aus. Kopiere den Text auf dieser Website und füge ihn an anderer Stelle ein. Am Ende müsste jetzt ein kurzer Lizenzhinweis stehen.
